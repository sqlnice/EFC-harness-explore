# EFC Harness Explore

This is a reveal.js presentation about using AI agents (Claude Code + Opus 4.6) to refactor a large enterprise frontend project (EFC, 9 micro-frontend sub-apps on qiankun). The talk covers the methodology of "Harness" — building automated feedback systems (tsc/eslint/vitest hooks) that let AI agents self-correct.

## Design Context

See [.impeccable.md](.impeccable.md) for full design context including tokens, principles, and aesthetic direction.

Key design principles for this presentation:
1. **数据先行，故事托底** — 大数字制造冲击，故事提供共鸣
2. **呼吸感优先于信息密度** — 投影场景，每页一个核心信息点
3. **暖色点亮，冷色打底** — 深色背景 + amber/orange 暖色用于人文节点，蓝色用于技术节点
4. **一致但不单调** — 各页面类型有视觉特征，共享色彩和间距
5. **为远距离观看设计** — 最小正文 22px，关键数字 64px+
